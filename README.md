## 🎮 Как запустить это безумие?

### 1️⃣ Установка зависимостей
Убедитесь, что у вас установлены все необходимые библиотеки:
```bash
pip install python-telegram-bot python-dotenv
```

### 2️⃣ Настройка секрета 🔑
Создайте (или отредактируйте) файл `.env` в корне проекта и вставьте туда ваш токен от BotFather:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

### 3️⃣ Поехали! 🚀
Запустите главный файл-переключатель:
```bash
python nemain.py
```

## 🛠️ Доступные интерфейсы

Мы подготовили три способа сойти с ума:

| Интерфейс | Описание | Команда |
| :--- | :--- | :--- |
| **🐚 Консоль (CLI)** | Олдскульный текстовый режим для настоящих хакеров. | Выбор "1" в `nemain.py` |
| **✨ Графика (GUI)** | Красивое темное окно на Windows. Стильно, модно, безумно. | Выбор "2" в `nemain.py` |
| **🤖 Телеграм Бот** | Анкета прямо в вашем мессенджере. Идеально для Telegram. | Выбор "3" в `nemain.py` |

## 📊 Где хранятся мои данные?

Все ваши ответы (и ответы ваших друзей/жертв) сохраняются в файл **`statistics.json`**. 
Там фиксируется:
- Текст ответов
- Выбранные варианты
- Время прохождения (timestamp)
